# デバッグツールのインストールや準備〜🎉

## 今からやること
- pryをインストール
- byebugをインストール

# 開発環境の準備
`git clone origin https://github.com/hiromuaraki/Pokemon.git` を実行

以降は上記でcloneしてきたフォルダ配下で作業します。

# pryのインストール

- `gem install pry pry-doc` を実行

# bey-bug

- `gem install pry-byebug` を実行

# 実行方法

- `pry` を実行

- `require './main' ` を実行

以降読み取りが可能となります。

以上で完了です🥂
